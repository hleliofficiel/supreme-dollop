Installation Guide

To begin using BebiHost, extract the archive and choose your desired platform folder: Main-Template, WordPress-Theme, or WHMCS-Template. Each folder contains production-ready code.

Project Structure

BebiHost/
├── Main-Template/       (Static HTML/CSS/JS)
├── WordPress-Theme/     (Custom WP Theme)
├── WHMCS-Template/      (WHMCS Client Area)
└── Documentation/       (This guide)

Development Guide

BebiHost strictly adheres to the principles defined in the Ultimate Web Design Agent guidelines. All templates share the exact same UI components, CSS (Nano UI & SuperGloss UI), and UX patterns to guarantee brand consistency.

WordPress Installation

1. Navigate to wp-content/themes/.
2. Upload the BebiHost/WordPress-Theme folder.
3. Go to WP Admin > Appearance > Themes and activate 'BebiHost'.

WHMCS Installation

1. Copy the BebiHost/WHMCS-Template folder to your WHMCS templates/ directory.
2. In WHMCS Admin, go to Setup > General Settings > General and select the new template.

CSS Guide (Nano UI)

The styling uses CSS variables for easy customization. The core aesthetic relies on glassmorphism (.glass-panel) and strict spacing variables.

:root {
  --primary-color: #0f172a;
  --secondary-color: #3b82f6;
  --glass-bg: rgba(255, 255, 255, 0.7);
}

Accessibility Guide

All components meet WCAG 2.1 AA standards. Color contrast ratios are verified, semantic HTML5 tags are used (header, main, footer, article, nav), and ARIA labels should be maintained when modifying templates.
